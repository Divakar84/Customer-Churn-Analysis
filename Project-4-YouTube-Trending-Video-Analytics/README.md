# Project 4: YouTube Trending Video Analytics

## 🎯 Objective
Uncover patterns in trending videos by analyzing YouTube datasets across regions.

## 🛠️ Tools Used
- **Python (Matplotlib, Seaborn):** Sentiment analysis and visualization.
- **SQL:** Data ranking and aggregation.
- **Tableau:** Time-series dashboards.

## 📋 Mini Guide
1. Clean and standardize YouTube trending datasets from different countries.
2. Perform sentiment analysis on titles and tags.
3. Use SQL to rank categories by avg views.
4. Create time-series visualizations for trending duration.

## 📦 Deliverables
- Dashboard: Most popular genres, sentiments.
- Region-wise comparison visuals.
- Final report with data storytelling.
