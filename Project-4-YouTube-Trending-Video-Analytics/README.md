# 📊 YouTube Trending Video Analytics

## 📌 Project Overview
This project analyzes trending videos on YouTube to uncover patterns in content popularity, engagement, and video performance. It focuses on identifying key factors that influence video virality using data analysis and visualization techniques.

---

## 🎯 Objectives
- Analyze trending video data across categories  
- Perform sentiment analysis on video titles  
- Evaluate engagement metrics (views, likes, comments)  
- Identify trends over time  
- Build an interactive dashboard using Power BI  

---

## 🛠️ Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn, TextBlob)  
- SQL  
- Power BI  

---

## 📁 Dataset
The dataset used in this project is not included in this repository due to file size limitations.

📥 You can download the dataset from:

After downloading, clean the dataset using the provided Python scripts.

## 🧹 Data Preprocessing
- Removed duplicates and corrupted rows  
- Handled missing values  
- Converted `trending_date` into proper datetime format  
- Converted numeric columns to appropriate data types  

---

## 💬 Sentiment Analysis
- Applied sentiment analysis on video titles  
- Classified into:
  - Positive  
  - Negative  
  - Neutral  

---

## 📊 Dashboard (Power BI)
The interactive dashboard includes:

- 📊 Category vs Views  
- 😊 Sentiment Distribution  
- 📈 Trending Over Time  
- 📉 Engagement Analysis (Views vs Likes)  
- 📋 Top Trending Videos  

---

## 📈 Key Metrics (KPIs)
- Total Views  
- Average Views  
- Total Likes  
- Engagement Rate  
- Viral Score  

---

## 🔍 Key Insights
- 🎬 Entertainment category dominates trending videos  
- 😊 Positive sentiment titles perform better  
- 🔥 Higher engagement leads to higher virality  
- 📈 Views show spikes during specific periods  
- ⏳ Most videos trend for a short duration
