# Project 7: Financial KPI Analysis for a Startup

## 🎯 Objective
Analyze monthly revenue, burn rate, CAC, LTV, and run rate for an early-stage startup.

## 🛠️ Tools Used
- **Excel:** Financial modeling and data collection.
- **Tableau:** Dashboard with trend indicators.
- **Python (Pandas):** Cohort analysis.

## 📋 Mini Guide
1. Collect financials: expenses, revenue, customer base.
2. Compute LTV:CAC ratio.
3. Build dashboard with trend indicators.
4. Perform cohort analysis (monthly customer groups).

## 📦 Deliverables
- Tableau dashboard
- LTV:CAC report in PDF
- Excel model template
