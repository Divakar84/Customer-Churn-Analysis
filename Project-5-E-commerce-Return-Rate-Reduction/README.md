# Project 5: E-commerce Return Rate Reduction Analysis

## 🎯 Objective
Identify why customers return products and how return rates vary by category, geography, and marketing channel.

## 🛠️ Tools Used
- **Python:** Logistic regression for return prediction.
- **Power BI:** Interactive risk score dashboards.
- **SQL:** Data cleaning and aggregation.

## 📋 Mini Guide
1. Clean return and order dataset.
2. Analyze return % per category and supplier.
3. Use logistic regression to predict probability of return.
4. Use Power BI to create return risk score dashboard.

## 📦 Deliverables
- Interactive dashboard with drill-through filters.
- Python codebase for prediction.
- CSV of high-risk products.
