# Project 8: Movie Success Prediction and Sentiment Study

## 🎯 Objective
Predict movie success using IMDB/Kaggle data, and analyze sentiment of viewer reviews.

## 🛠️ Tools Used
- **Python (NLTK, VADER, Sklearn):** Sentiment analysis and predictive modeling.
- **Excel:** Data organization and reporting.

## 📋 Mini Guide
1. Scrape or import IMDB movie + rating data.
2. Use VADER for sentiment on user reviews.
3. Create regression model to predict box office success.
4. Analyze genre-wise sentiment trends.

## 📦 Deliverables
- Python notebooks
- Sentiment visuals
- Predictive model summary
